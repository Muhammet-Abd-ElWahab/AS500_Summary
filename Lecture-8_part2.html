üìå Introduction to Normal Distribution

The normal distribution is the most important probability distribution in statistics. It appears naturally in many real-world phenomena and serves as the foundation for many statistical tests and procedures.

Why is Normal Distribution Important?

  • Many natural phenomena follow normal distribution (heights, weights, test scores)
  • Central Limit Theorem: sample means approach normal distribution
  • Foundation for hypothesis testing and confidence intervals
  • Simplifies statistical calculations and predictions

Historical Context

As sample sizes increase, the binomial distribution approaches a normal distribution. This is why the normal distribution is so fundamental to statistics.

  • 50% of the data lies on either side of the mean
  • The shape resembles a bell, hence "bell curve"

This allows us to standardize and compare different normal distributions.

‚≠ê The Standard Normal Distribution

Standard Normal Distribution

The standard normal distribution is a special normal distribution with:

  • Mean (Œº) = 0
  • Standard Deviation (œÉ) = 1
  • Notation: Z ~ N(0, 1)

Why Standard Normal?

Advantages

  • Any normal distribution can be converted to standard normal
  • Standard normal tables are readily available
  • Simplifies probability calculations
  • Allows comparison across different scales

Standard Normal Table

üìä The Empirical Rule (68-95-99.7 Rule)

The Empirical Rule

For any normal distribution:

  • 68% of data falls within ¬±1œÉ of Œº
  • 95% of data falls within ¬±2œÉ of Œº
  • 99.7% of data falls within ¬±3œÉ of Œº

  • The middle 68% is between Œº - œÉ and Œº + œÉ
  • The middle 95% is between Œº - 2œÉ and Œº + 2œÉ

Given: IQ scores ~ N(100, 15¬≤)

Interpretation:

  • 68% of people: IQ between 85 and 115 (100 ¬± 15)
  • 95% of people: IQ between 70 and 130 (100 ¬± 30)
  • 99.7% of people: IQ between 55 and 145 (100 ¬± 45)

Practical Applications

  • Quality control in manufacturing

üìã Common Probability Calculations

Type 1: P(Z < a)

If a > 0: P(Z < a) = 0.5 + P(0 < Z < a)
If a < 0: P(Z < a) = 0.5 - P(0 < Z < |a|)

Type 2: P(Z > a)

P(Z > a) = 1 - P(Z < a)

Type 3: P(a < Z < b)

P(a < Z < b) = P(Z < b) - P(Z < a)

Example: P(1 < Z < 2)

  • P(Z < 2) = 0.5 + 0.4772 = 0.9772
  • P(Z < 1) = 0.5 + 0.3413 = 0.8413
  • P(1 < Z < 2) = 0.9772 - 0.8413 = 0.1359

Example: P(Z < -2.47)

  • From table: P(0 < Z < 2.47) = 0.4932
  • By symmetry: P(Z < -2.47) = 0.5 - 0.4932 = 0.0068
