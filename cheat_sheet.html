üìã Sheet 1: Statistics Basics - Quick Tips

üéØ Key Distinction: Descriptive vs Inferential

TRICK: Ask "Are we describing what we have OR predicting about the population?"

  • DESCRIPTIVE: "Ahmed's average is 85" ‚Üí We're describing the data we have
  • INFERENTIAL: "Ahmed will score higher next time" ‚Üí We're predicting/inferring

üìã Sheet 2: Data Types & Sampling - Quick Tips

üéØ Data Type Decision Tree

TRICK: Follow this path to identify data types

  1. Can it be counted?
    • YES ‚Üí DISCRETE (number of students, accidents, etc.)
    • NO ‚Üí CONTINUOUS (height, weight, temperature)
  2. Is it categorical?
    • YES ‚Üí QUALITATIVE (gender, nationality, color)
    • NO ‚Üí QUANTITATIVE (age, income, height)

üéØ Measurement Level Decision Tree

TRICK: Remember "NOIR" - Nominal, Ordinal, Interval, Ratio

  • NOMINAL: Categories with NO order (Gender, Nationality, Color)
    • ‚úì Can only count frequencies
  • ORDINAL: Categories WITH order (Grades: A, B, C; Satisfaction: Low, Medium, High)
    • ‚úì Can rank but no meaningful differences
  • INTERVAL: Numeric with NO true zero (Temperature in Celsius, IQ scores)
    • ‚úì Can add/subtract but not multiply/divide
  • RATIO: Numeric WITH true zero (Age, Income, Height, Weight)
    • ‚úì Can do all operations including ratios

üéØ Sampling Methods Quick Reference

TRICK: Match the sampling method to the situation

  • SIMPLE RANDOM: Each item has equal chance ‚Üí Lottery, random numbers
  • STRATIFIED: Population divided into groups ‚Üí Sample from each group proportionally
  • SYSTEMATIC: Every kth item ‚Üí 1 in 10 sampling
  • JUDGMENT: Researcher chooses ‚Üí NOT representative

‚ö° Stratified Sampling Calculation

Formula: Sample size from stratum = (Stratum size / Total population) √ó Total sample size
Example: 300 females, 500 males, total 800. Want sample of 80.
Females: (300/800) √ó 80 = 30
Males: (500/800) √ó 80 = 50

üìã Sheet 4: Probability - Quick Tips

üéØ Probability Calculation Quick Reference

TRICK: Use this formula for basic probability

  • P(A) = Favorable outcomes / Total outcomes
  • P(A') = 1 - P(A) (Complement)
  • P(A or B) = P(A) + P(B) - P(A and B) (Addition)
  • P(A and B) = P(A) √ó P(B) (Multiplication for independent)

üéØ Independent vs Dependent Events

TRICK: Ask "Does first event change second event's probability?"

  • INDEPENDENT: NO change
    • ‚úì Coin tosses, die rolls, WITH replacement
    • ‚úì P(A|B) = P(A)
  • DEPENDENT: YES change
    • ‚úì Drawing WITHOUT replacement, related events
    • ‚úì P(A|B) ‚â† P(A)

‚ö° Mutually Exclusive Events

TRICK: If events cannot happen together ‚Üí Mutually Exclusive

Formula: P(A ‚à© B) = 0

üìè Measurement Level Quick Guide (NOIR)

üî¥ NOMINAL - Categories, No Order

Characteristics:

  • Only counting frequencies
  • No meaningful order
  • Cannot perform math operations

Examples:

  • Gender (Male, Female)
  • Nationality (Egyptian, American, etc.)
  • Color (Red, Blue, Green)
  • Religion (Muslim, Christian, etc.)

üü† ORDINAL - Categories, With Order

Characteristics:

  • Can rank/order
  • Differences not meaningful
  • Cannot perform math operations

Examples:

  • Grades (A, B, C, D, F)
  • Satisfaction (Very Satisfied, Satisfied, Neutral, Dissatisfied)
  • Education Level (High School, Bachelor, Master, PhD)
  • Rank (1st, 2nd, 3rd)

üü° INTERVAL - Numeric, No True Zero

Characteristics:

  • Can add/subtract
  • No true zero point
  • Cannot multiply/divide meaningfully

Examples:

  • Temperature in Celsius (0¬∞C is arbitrary)
  • IQ Scores
  • Calendar Years

üü¢ RATIO - Numeric, True Zero

Characteristics:

  • Can do all operations
  • True zero point exists
  • Ratios are meaningful

Examples:

  • Age (0 = not born yet)
  • Income in dollars (0 = no income)
  • Height in cm (0 = no height)
  • Weight in kg (0 = no weight)
  • Number of students (0 = no students)
