üìå Random Variables

What is a Random Variable?

A random variable is a function that assigns a single numeric value to each outcome in a sample space. It converts experimental outcomes into numbers.

Example: Four Births

Consider 4 sequential births. Let X = number of girls.

  • BBBB ‚Üí X = 0
  • BBBG, BBGB, BGBB, GBBB ‚Üí X = 1
  • BBGG, BGBG, BGGB, GBGB, GBBG, GGBB ‚Üí X = 2
  • BGGG, GBGG, GGBG, GGGB ‚Üí X = 3
  • GGGG ‚Üí X = 4

üîÄ Discrete vs Continuous Random Variables

Discrete Random Variables

Characteristics

  • Has a countable number of possible values
  • Has discrete jumps (gaps) between successive values
  • Has measurable probability for each individual value
  • Counts things

Examples

  • Number of defective items in a batch
  • Number of customers arriving per hour
  • Number of heads in 10 coin tosses
  • Number of students in a class

Continuous Random Variables

Characteristics

  • Has an uncountably infinite number of possible values
  • Moves continuously from value to value (no gaps)
  • Has NO measurable probability for individual values
  • Measures things

Examples

  • Height of a person
  • Weight of an object
  • Time to complete a task
  • Temperature

Key Difference

Aspect Discrete Continuous
Values Countable (1, 2, 3...) Infinite (any value in range)
Gaps Yes (jumps between values) No (smooth transitions)
P(X = specific value) Can be > 0 Always = 0
Measures Counts Measurements

The expected value (or mean) is the average value of a random variable, weighted by probabilities. It's what we expect to get on average if we repeat the experiment many times.

üé≤ Bernoulli Distribution

Definition

A Bernoulli trial is an experiment with only two possible outcomes: success or failure. A Bernoulli random variable counts the number of successes in one trial (0 or 1).

Bernoulli Distribution Formula

P(X = 1) = p (probability of success)
P(X = 0) = 1 - p = q (probability of failure)

Mean and Variance

E(X) = p
V(X) = p(1 - p) = pq

Example: Fair Coin Toss

  • Success = Heads, p = 0.5
  • Failure = Tails, q = 0.5
  • E(X) = 0.5
  • V(X) = 0.5 √ó 0.5 = 0.25

Applications

  • Medical: Disease present/absent
  • Quality control: Defective/non-defective
  • Marketing: Purchase/no purchase

üìä Binomial Distribution

Definition

The binomial distribution counts the number of successes in n independent Bernoulli trials.

Conditions for Binomial Distribution

  • Fixed number of trials (n)
  • Each trial has two outcomes (success/failure)
  • Probability of success (p) is constant
  • Trials are independent

Binomial Probability Formula

P(X = x) = C(n,x) √ó p^x √ó q^(n-x)
where C(n,x) = n! / (x!(n-x)!)

Example: Five Coin Tosses

What is the probability of exactly 2 heads in 5 tosses?

  • n = 5, x = 2, p = 0.5, q = 0.5
  • C(5,2) = 5! / (2! √ó 3!) = 10
  • P(X = 2) = 10 √ó (0.5)¬≤ √ó (0.5)¬≥ = 10 √ó 0.03125 = 0.3125

üîî Poisson Distribution

Definition

The Poisson distribution counts the number of occurrences of an event over a fixed time period or space, when events occur independently at a constant average rate.

When to Use Poisson

  • Counting events in a fixed interval (time, space, area)
  • Events occur independently
  • Average rate is constant
  • Can approximate binomial when p is small and n is large

Poisson Probability Formula

P(X = x) = (e^(-Œª) √ó Œª^x) / x!
where Œª = average rate, e ‚âà 2.718

Mean and Variance of Poisson

E(X) = Œª
V(X) = Œª
(Mean equals variance!)

Example: Call Center

A call center receives 10 calls per hour on average. What is the probability of receiving exactly 3 calls in an hour?

  • Œª = 10
  • P(X = 3) = (e^(-10) √ó 10¬≥) / 3!
  • P(X = 3) = (0.0000454 √ó 1000) / 6 ‚âà 0.0076

Applications

  • Number of customer arrivals per hour
  • Number of defects in a production batch
  • Number of accidents per month
  • Number of emails received per day

üéØ Geometric Distribution

Definition

The geometric distribution counts the number of trials needed to get the first success in a sequence of independent Bernoulli trials.

Geometric Probability Formula

P(X = n) = (1 - p)^(n-1) √ó p
where n = number of trials until first success

Mean of Geometric Distribution

E(X) = 1/p

Example: Rolling a Die

What is the probability of rolling a 3 on the 5th roll (not before)?

  • p = 1/6, n = 5
  • P(X = 5) = (5/6)‚Å¥ √ó (1/6) = 0.0804
  • Expected number of rolls: E(X) = 1/(1/6) = 6

‚Ä¢ k = trial number where FIRST success occurs
‚Ä¢ X = number of trials needed to get first success

‚úÖ Key Takeaways

  • Random variables assign numeric values to experimental outcomes
  • Discrete variables count; continuous variables measure
  • Probability distributions list all values and their probabilities
  • Expected value is the weighted average of all outcomes
  • Variance measures spread; standard deviation is its square root
  • Bernoulli: single trial with two outcomes
  • Binomial: number of successes in n independent trials
  • Poisson: number of events in fixed interval (mean = variance)
  • Geometric: number of trials until first success
  • Choose distribution based on data characteristics and context
